# Projet SIN x CIRA

L'objectif du projet est de proposer une solution permettant de mesurer le niveau de remplissage d'une cuve à l'aide d'un capteur à ultrason placé au sommet de la cuve.

### Matériel utilisé

-   1 Sonde de mesure Vega EK21
-   1 Carte Arduino UNO
-   1 Module Bluetooth
-   1 Afficheur LCD
